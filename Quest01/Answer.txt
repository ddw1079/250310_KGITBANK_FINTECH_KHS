# 문제 1.
    리눅스 종류와 특징에 대해 조사하여 작성하고 그중에서 우분투리눅스에 대해 상세하게 작성하시오.

1. 리눅스의 종류
Debian Linux (데비안 리눅스) :데비안 프로젝트가 개발한 자유 컴퓨터 운영체제

- Ubuntu Linux (우분투 리눅스) :영국 기업 캐노니컬이 개발,배포하는 컴퓨터 운영체제
                            데비안 리눅스 기반으로 개발.(데비안에 비해 사용편의성에 초점을 맞춘 리눅스 배포판)
                            일반적으로 매 6개월마다 새로운버전 공개.


- RedHat Linux (레드헷 리눅스) :미국에 본부를 둔 레드햇사가 개발하던 리눅스 배포판.
                            서버 안정성이 높음/설치 용이/폭넓은 하드웨어 지원/대규모 데이터베이스 환경에서도 안정적으로 동작
                            (최신기술 사용 시 유료 지원_유료버전 사용 시 지속적 관리와 최신기술지원, 보안업데이트 등 지원)

- Fedora Linux(페도라 리눅스) : 리눅스 커널 개발자 사용 운영체제/전문가를 위한 리눅스 
                            레드햇에서 개발되는 기술 가장 먼저 탑재되어 최신기술 가장 먼저 사용 가능
                            개발 위한 도구들 기본 제공 (에러 많음)

- Centos(센트OS) :centOS 프로젝트에서 레드햇 제휴로 개발한 컴퓨터 운영 체제.
                 전세계 점유율 1위/무료 사용가능/가볍고 안정적 운영 가능.

- Kali Linux(칼리 리눅스) : offensive Security가 개발한 컴퓨터 운영 체제.
                        (데비안 기반의 리눅수 운영체제/칼리 리눅스 안 백트랙처럼 수많은 해킹 관련 도구,설명서 포함)
                        백트랙 후속버전으로 해킹 및 정보보안을 테스트 하기 위해 만들어짐. 무료 다운로드 가능
                        배포되고 있는 리눅스 서버 중 안정성 가장 높으며 포함하고있는 패키지도 다양.
                        (유명소프트웨어와 상용 불가능, 전문지식 요구)
리눅스에는 많은 종류와 그 파생 종류가 있지만 대표적으로 Debian 타입과 Fedora 타입을 예로 들 수 있고,
